Context manager for limiting in the time the execution of a block using asynchronous threads launching TimeoutError exception.

The Timeout context manager allows you to specify a time limit for a block of code. If the block does not complete within the specified time limit, a TimeoutError exception is raised in the thread executing the block.

..code-block:: python3
linenos:

caption:

Example of using the Timeout context manager. try:

with Timeout(timeout_interval=5.0) as timeout:

# Code block that should complete within 5 seconds do_something()

if timeout.state == TimeoutState.EXECUTED:

except TimeoutError:
